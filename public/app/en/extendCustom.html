Extend and Customize

Optional: What's Next?

Now that you tried the out-of-the box version of the solution you can start extending it to meet your needs. Look at the following suggestions to further familiarize yourself with Node-RED and to use your new skills to use these tools to build your own IoT Solution..

Extend the Existing Solution

To add humidity data to the simulation tab of the node-RED instance:

  1. Click Edit Flows in Node-RED button.
  2. Select the Virtual Thermostat tab and edit the device schema to include data for humidity.
  3. Copy and paste the temperature data, and replace the word temperature with humidity.
  4. Modify the integer values to be in a range of 0 to 100.
  5. Click Done in the node editor and then Done to deploy your node flow.
  6. On the IoT for Climate control landing page, click View simulator to see the new humidity data.
  7. Optional: Modify the prototype app to include humidity alerts and alarms.

Customize Your Own Solution

Customize this solution instance to fit your own intended solution:

  1. Use Watson IoT Platform to create your own device types.
  2. Edit and deploy your own virtual device simulation flow in Node-RED.
  3. Use the simulator to create multiple instances of your virtual devices.
  4. Build your own proof-of-concept dashboard with node-RED.
