Sample IoT Solution for Climate Control

The thermostat device type is registered in IBM Watson IoT Platform. Use the virtual climate control simulation flow to generate random temperature data for your thermostats.

  1. Use the Mobile or Desktop link to open the simulator.
  2. In the simulation dashboard, enter or select the number of virtual thermostats that you want to create..
  3. Click Create.
  4. Verify that your new virtual thermostats start. If require click the Start/Stop slider to start the thermostat.
  5. Click the edit icon for each thermostat to update the actual and desired temperatures as needed, then click Save.
