Prototype Application

Climate Control Alert and Email App

Use the Climate Control end-user app to set email notifications that are based on room temperatures. Use the end-user app to: Monitor and control a particular room, set the email address for a service engineer, and change the alert thresholds. The app is built by using the Node-RED flows that control the monitoring and alerting UI. You can customize the look, feel, and behavior of the app by modifying the flows and nodes that control the monitoring and alerting UI.

  1. Click View App.
  2. Use the Alert Configuration section of the UI to set alarm thresholds and email address.
  3. Use the Thermostat section of the UI to select a room within the store to monitor the corresponding thermostat.
  4. Back on the IoT for Climate Control landing page, click Edit Flows in Node-RED.
  5. Examine the Monitoring Flow tab that controls the data flow from Watson IoT Platform into the prototype application.
  6. Examine the Alert Configuration tab that generates the UI controls for the alert configuration in the prototype app.
  7. Back on the IoT for Climate Control landing page, click Next.

For more information, see the documentation for the dashboard and Watson IoT manager nodes.
