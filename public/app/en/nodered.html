Node-RED

Description

Node-RED is a visual wiring tool for the Internet of Things. Use its browser-based flow editor to wire together hardware devices, APIs, and online services in new and interesting ways by using a wide ranging palette of flow nodes.

Sample IoT Solution for Climate Control

After creating the climate control device type in Watson IoT Platform, you can use the pre-generated Node-RED flow to generate device data for a thermostat.

  1. Click Go to Node-RED Flow Editor.
  2. Inspect the different simulation nodes on the Virtual Thermostat tab.
  3. Deploy the virtual thermostat tab by clicking the red Deploy button.
  4. Back on the IoT for Climate Control landing page, click Next.

Go to Node-RED flow editor

Simulator Nodes

Use the six nodes in the simulator category to simulate the behavior of your solution devices.

Device Manager Nodes

Use the device manager nodes for Watson IoT Platform device operations such as: Register, Delete, Retrieve, and Update device types and instances.

Form UI Node

This node is a component of the Node-RED Dashboard UI. It is used to create forms in the Node-RED Dashboard UI.
